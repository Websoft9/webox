# PRD

## 概要

- 产品名称：应用聚合与托管平台
- 价值体现：聚合/接入/运行/连接一切应用，让用户每天上班第一件是就是打开 Websoft9 控制台，开启一天轻松基于 Web 随时随地办公模式
- 业务功能：寻找、部署、接入、运行、发布、管理、安全
- 技术功能：多用户多权限、Vault、Git、SSO

## 业务需求

### 寻找

基于 AI + 知识库，根据用户需求推荐具体的软件集。

### 部署

将软件部署到目标服务器：

- 多服务器支持
- 目标服务器环境准备
- Docker/Podman 容器引擎支持
- 部署编排模式
  - docker-compose
  - docker run
  - 基于源码构建镜像
- 部署流程可视化
- 异步
- 部署物准备
  - 镜像
  - 软件仓库
  - Websoft9 制品库
- 私有云
